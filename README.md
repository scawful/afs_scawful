# AFS Scawful

**Personal Skills & Context for AFS.**

This repository contains personal configurations, skills, and context data for the Agentic File System (AFS). It serves as a personal "soul" or extension of the core AFS, tailoring the agent's behavior and knowledge to my specific workflows and projects.

## Contents

- **Skills:** specialized capabilities for agents (e.g., specific coding styles, project management).
- **Context:** Personal documentation, notes, and memory files.
- **Config:** Custom configurations for AFS services and tools.
- **Policies:** Guidelines and rules for agent behavior.

## Usage

This repository is intended to be mounted or linked into an AFS installation to provide personalized context and capabilities.
